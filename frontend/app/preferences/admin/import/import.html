Import

List accounts all at once by importing a .csv file containing account information.

  • Only .CSV files are eligable
  • The file should at least contain a column with the header: name
  • Optional headers are: website, email address, phone number, twitter, address, postal code, city
  • Information in any additional columns will be added to the description of an account
  • Please note that headers are case sensitve
  • {{ name }}
  • {{ name }}
